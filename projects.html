Optum:

During my internship with Optum this past summer, 2022, I had the privillage of working with the Microsoft platform. My team was given the job of putting together a system to take information from a Microsoft SharePoint list, to a SQL table in the Optum database. To do this, the team used Power Automate, a Microsoft application that allows connections between Microsoft softwares to be done automatically. By using Power Automate, the team was able to draw all the list information in the Microsoft SharePoint, then copy that information to an SQL table in the database. It happened automatically, which means when there was an update to the Microsoft SharePoint, the SQL table would also get updated, mimicing the Sharepoint list perfectly. This project allowed the active Microsoft Sharepoint list to exist in an SQL table, that way other services that create reports can pull the SQL table information, and render the report with the new information. This was not possible to gather from Microsoft Sharepoint automatically.



CISC 420:

Last spring, I had the opprotunity to take a computer science elective, and I chose CISC 420, Computer Graphics. Later in the semester, we started to work on a project that worked on lighting objects in a three dimensional world, displayed in two dimensions. There was Gouraud lighting and Phong lighting. The image on the left shows a octagonal prism and a sphere in space, with three lighting sources currently selected in the scene. The lighting affects each object based on the light source and the color. Also using the W, A, S, and D keys on the keyboard allow you to move the scene around. Using the left and right arrow keys rotate the scene, making it possible for you to navigate any rendered scene with the light sources!
